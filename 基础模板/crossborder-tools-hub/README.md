# 跨境电商数据洞察补给站

## 项目概述

这是一个专为跨境电商运营人员设计的数据分析工具集合网站，旨在补充现有工具（如Jungle Scout、Helium 10）无法覆盖的功能盲区。

**🌐 网站地址：** https://nqf46zs3prbn.space.minimax.io

## 核心特性

### 🎯 基于简介的工具发现
- **无需搜索**：通过简洁的工具卡片设计，让用户快速了解每个工具的功能
- **清晰分类**：按功能类别分组（广告优化、库存管理、评论分析等）
- **视觉引导**：使用颜色编码和图标系统，帮助用户快速识别工具类型

### 🔄 个性化排序系统
- **拖拽排序**：点击"调整顺序"进入自定义模式，支持拖拽重新排列工具
- **本地保存**：排序偏好自动保存到浏览器本地存储
- **独立体验**：个人排序设置不影响其他用户的使用体验
- **一键重置**：支持快速恢复默认排序

### 📱 现代化用户体验
- **响应式设计**：完美适配桌面、平板、手机等各种设备
- **流畅动画**：悬停效果、拖拽反馈、状态切换都有精心设计的动画
- **直观界面**：极简SaaS风格设计，减少视觉干扰，提升专注度

## 工具分类

### 🔍 数据分析类
- **关键词挖掘助手** - 基于竞品评论挖掘长尾关键词
- **评论情感分析器** - 深度分析产品评论情感倾向
- **季节性预测器** - 基于历史数据预测产品季节性需求
- **转化率优化器** - 分析Listing要素对转化率的影响

### 📦 运营管理类
- **库存预警计算器** - 智能预测断货时间
- **物流成本优化** - 计算最优发货策略
- **税务合规检查** - 自动检测跨境销售税务风险
- **汇率波动预警** - 实时监控货币汇率变化

### 🎯 竞争分析类
- **竞品定价监控** - 实时追踪竞品价格变动
- **广告词频统计** - 分析竞品广告文案高频词汇
- **产品生命周期分析** - 判断产品所处生命周期阶段

## 技术实现

### 前端技术栈
- **React 18** + **TypeScript** - 现代化前端框架
- **Vite** - 快速构建工具
- **Tailwind CSS** - 实用优先的CSS框架
- **@dnd-kit** - 专业的拖拽功能实现

### 核心功能实现
- **拖拽排序**：使用@dnd-kit库实现流畅的拖拽体验
- **状态管理**：React Hooks管理组件状态
- **数据持久化**：LocalStorage保存用户偏好
- **响应式布局**：CSS Grid + Flexbox实现自适应布局

## 使用说明

### 基础操作
1. **浏览工具**：在首页查看所有可用工具
2. **分类筛选**：点击顶部分类标签筛选特定类型工具
3. **查看详情**：悬停或点击工具卡片了解详细信息

### 自定义排序
1. **进入自定义模式**：点击右上角"调整顺序"按钮
2. **拖拽重排**：拖拽工具卡片到理想位置
3. **保存设置**：排序自动保存，下次访问时保持
4. **重置排序**：点击"重置默认"恢复初始顺序

### 响应式使用
- **桌面端**：4列网格布局，完整功能展示
- **平板端**：3列网格布局，适配触摸操作
- **手机端**：单列布局，优化移动端体验
