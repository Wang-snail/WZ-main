# 智策 - 数据驱动产品战略规划平台

## 项目概述

"智策"是一个基于权威数据源的产品战略规划平台，旨在为产品经理和战略规划师提供数据驱动的决策支持系统。该平台集成了多个公信力数据平台，通过智能分析算法，帮助用户发现市场机会、制定产品策略、优化资源配置。

## ✨ 核心功能

### 1. 数据驱动仪表盘
- 实时KPI监控：市场增长指数、竞争强度指数、用户需求指数、产品机会评分
- 智能洞察系统：基于数据模式自动识别机会和风险
- 可视化图表：趋势分析、竞争态势、需求分析、产品组合分析

### 2. 权威数据源配置
- **Google Trends API**: 全球搜索趋势，180+国家，5年历史数据
- **百度指数**: 中文搜索行为数据，实时热度追踪
- **世界银行开放数据**: 全球经济发展指标，完整历史数据
- **国家统计局**: 中国官方宏观经济数据
- **Statista数据库**: 22,500+来源的行业报告
- **社交媒体监测**: Twitter、微博、知乎情感分析

### 3. BCG产品组合矩阵
- 智能产品分类：基于市场份额和增长率自动分析
- 四象限策略：明星产品、金牛产品、问题产品、瘦狗产品
- ROI分析：投资回报计算和资源分配建议
- 战略建议：差异化竞争策略指导

### 4. 产品路线图规划
- 可视化时间轴：甘特图展示项目进度
- 优先级管理：高、中、低优先级智能排序
- 依赖关系：项目间依赖和风险评估
- 工作量评估：人天计算和资源配置

### 5. 市场机会分析
- 多维度评估：市场规模、增长率、竞争强度、进入壁垒
- ROI预测：数据模型驱动的投资回报预测
- 趋势识别：市场机会和威胁早期预警
- 风险评估：技术、市场、政策风险量化

## 🚀 快速开始

### 本地开发
```bash
# 克隆项目
git clone <repository-url>
cd product-planning-platform

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```

### 项目结构
```
src/
├── components/          # React组件
│   ├── Dashboard.tsx    # 仪表盘
│   ├── DataSourceConfig.tsx  # 数据源配置
│   ├── BCGMatrix.tsx    # BCG矩阵分析
│   ├── Roadmap.tsx      # 产品路线图
│   └── MarketAnalysis.tsx # 市场机会分析
├── hooks/              # 自定义Hooks
├── lib/                # 工具函数
└── App.tsx             # 主应用组件
```
