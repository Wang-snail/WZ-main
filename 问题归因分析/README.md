# RootCause AI - 智能归因分析平台

## 项目概述

RootCause AI是一个智能化的根因分析网页应用，旨在帮助用户系统性地分析问题、找到根本原因并生成针对性的解决方案。该平台采用现代化的Web技术栈，提供直观易用的用户界面和强大的分析功能。

## 🎯 核心功能

### 1. 问题定义模块
- **功能**：收集问题的基本信息，包括标题、分类、时间、地点、影响范围等
- **特点**：提供分类模板，智能表单验证，帮助用户结构化问题描述

### 2. 5 Why 根因分析
- **功能**：通过连续追问"为什么"，深入挖掘问题的根本原因
- **特点**：
  - 智能问题模板推荐
  - 步骤化引导分析
  - 根因标记功能
  - 实时进度跟踪

### 3. 鱼骨图分析
- **功能**：从多个维度（人、机、料、法、环、测）系统分析问题原因
- **特点**：
  - 交互式可视化鱼骨图
  - 6个分析维度的预设模板
  - 置信度评估
  - 拖拽式原因添加

### 4. 智能解决方案推荐
- **功能**：基于根因分析结果，生成针对性的解决方案
- **特点**：
  - 行业模板库支持
  - 优先级和投入成本评估
  - 解决方案选择和定制
  - 实施建议

### 5. 报告导出
- **功能**：生成完整的分析报告，支持多种格式导出
- **特点**：
  - Markdown格式报告
  - 结构化内容组织
  - 解决方案优先级排序
  - 实施建议和风险控制
