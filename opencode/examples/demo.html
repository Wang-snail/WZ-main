📝 集成代码示例

JavaScript 集成： 
<script src="opencode-sdk.js"></script>
<script>
  const sdk = new OpenCodeWebSDK({ mode: 'static' });
  await sdk.loadKnowledge('./knowledge_snapshot.json');
  const result = sdk.queryKnowledge('CR规则');
  console.log(result);
</script>
Python 集成： 
from opencode_sdk import OpenCodePythonSDK

sdk = OpenCodePythonSDK('./knowledge_snapshot.json')
result = sdk.query_knowledge('CR规则')
print(result)
